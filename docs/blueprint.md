# **App Name**: Baalvion Public Investor Relations Website Prototype

## Core Features:

- Public Information Display: Displays all public website pages and sections, including Home, Company Overview, Investment Thesis, Governance & Leadership, Risk & Compliance, Media/News, and Contact, populated with realistic mock data.
- Investor Registration Flow Prototype: Provides a clickable, multi-step user interface for investor onboarding, including account creation, email verification, multi-factor authentication, basic profile management, accreditation self-certification, and NDA e-sign placeholders.
- Dynamic UI Interactions & CTAs: Implements interactive elements such as a sticky header with a color transition on scroll, and configures Call-to-Action buttons where 'Apply for Investment Access' opens the registration modal, and 'Learn More' smooth-scrolls to the Company Overview section.
- Responsive Layout System: Ensures the application is fully responsive, built with a mobile-first approach using Tailwind CSS, providing an optimal and consistent viewing experience across all device sizes (mobile, tablet, desktop).
- Mock Data Integration & Simulation: Simulates backend API calls and integrates mock data for all content, including news articles, executive bios, charts, and user account information, to create a fully populated and realistic frontend experience without a real database.
- Multi-language Toggle Placeholder: Includes a non-functional user interface element for switching between different languages, demonstrating future internationalization capabilities.
- Subtle UI Animations: Incorporates subtle visual transitions and animations for elements like the sticky header, modals, and navigation changes to enhance user experience and provide a polished, premium feel.

## Style Guidelines:

- The layout will be spacious, modular, and feature clearly delineated sections to enhance readability and ensure content is presented in an organized, digestible manner.
- The 'Inter' font (sans-serif) will be used for all headings and body text, chosen for its modern, clean, and highly legible appearance across all content types.
- Minimalist, system-style icons will be used throughout the application to maintain a clean, modern aesthetic and improve clarity without adding visual clutter.
- Subtle transitions will be applied to the sticky header, modals, and navigation changes to enhance the overall user experience.
- Primary color: A deep, authoritative navy blue (#1A2B42) to convey stability, trust, and corporate strength, suitable for backgrounds and main elements.
- Accent color: A sophisticated gold or amber (#CBA135) for highlights, interactive elements, and key call-to-actions, symbolizing value and importance.
- Text color: A light, off-white or very light grey (#F0F2F5) for optimal readability against darker backgrounds, maintaining a clean aesthetic.
- Neutral background: A very dark grey (#0F172A) to provide depth and contrast, serving as a base for various sections and maintaining a premium feel.
