## Overview

The **Baalvion Investor Relations** site is the official channel for the group's communications with
investors and the market — press releases, financial news, and corporate updates, served from a
managed content system.

## ✨ Features

- **Press releases** — official announcements and disclosures
- **Financial news** — updates and corporate milestones
- **Investor resources** — communications and key information
- **CMS-driven** — content managed centrally, delivered through a public API
